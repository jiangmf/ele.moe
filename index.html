projects

osu!graphs

osu!graphs is a statistics site for the online rhythm game - "osu!" built on python Django. It fetches player data using a RESTful API on an hourly basis and stores it a MySQL Database. It then perform analysis on said data to find trends and patterns, and presents it to the user with intuitive graphs using JavaScript.

Web Django Python HTML JavaScript CSS

Magic Tower

Magic Tower is a simple puzzle RPG game built using C++ and OpenGL. The player navigates through the tower collecting power-ups and defeating monsters. The game uses models created using Blender and graphics created with Photoshop.

Game C++ OpenGL Blender Photoshop

Ice Lrc

Ice Lrc is a tool that goes through your music library and tries to find the matching lyrics using mutagen for ID3 info, and Google search API and Beautiful soup for web search and scraping. It then embed the lyrics to the audio file itself for easy access by your favorite media player.

Software Python Beautiful Soup
